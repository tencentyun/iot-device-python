## 引用方式

**引用方式**

-  Install from pip

    ```
	pip install tencent-iot-device
	```


-  Build from source

	```
	git clone https://github.com/tencentyun/iot-device-python.git
	cd iot-device-python
	python setup.py install
	```


**Python SDK源码**

-  如果您想通过代码集成方式进行项目开发，可访问[Github](../)下载python sdk源码。
